Einstellungen

Anleitung: Um diese Erweiterung nutzen zu können, benötigen Sie einen Persönlichen Zugriffstoken von Asana.

  1. Besuchen Sie Asana Access Token Seite
  2. Klicken Sie auf "Neues Token erstellen"
  3. Geben Sie einen Namen für das Token ein, z.B. "WPWW Task Creator"
  4. Kopieren Sie das generierte Token (wird nur einmal angezeigt!)
Ihr Token wird nur lokal in Ihrer Browser-Erweiterung gespeichert.
